# Clojure Brainfuck

A small interpreter written in Clojure for the toy language [https://en.wikipedia.org/wiki/Brainfuck](Brainfuck).

## Usage

    $ lein run [filename]

## Examples

A helloworld.bf example is provided in resources. This prints the string "Hello World!".
